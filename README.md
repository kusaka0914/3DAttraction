■ゲームの概要

【ゲームタイトル】Slime's Sky Travel

【ジャンル】アドベンチャー

【特徴】空中に浮かぶ足場から足場へとジャンプで渡り歩き、ステージ上に散らばった3つのアイテムをすべて集めてゴールを目指す、アトラクション感覚のアクションゲームです。
本作の特徴は、ゲームプレイ中に [T]キー を押すことでプレイヤーの移動速度を上昇させられる点です。スピードアップ中は爽快感が増す一方で、ステージ上のギミックも同時に加速するため、難易度がぐっと上がります。さらに、プレイヤー自身の操作もシビアになり、繊細なコントロールが求められます。
クリアタイムに応じて獲得できる星の数が変化するため、可能な限り速いスピードに設定しながら、高速で動くギミックを華麗にかわし、最速クリアを目指してください。
一度クリアしたステージではタイムアタックを行うことも可能です。何度もプレイしてステージを極め、最速クリアを更新してください。

【プレイ時間】10分～30分程度

【ゲーム制作者】日下　拓海

■ゲームの起動

windows

本ページ右側にあるReleasesから最新版のSlimesSkyTravel_win_v~.zipをインストールします。

そのzipファイルを展開し、「start_game.bat」ファイルをダブルクリックしてください。

mac

本ページ右側にあるReleasesから最新版のSlimesSkyTravel_mac_v~.zipをインストールします。

そのzipファイルを展開し、「start_game.sh」ファイルをダブルクリックしてください。
